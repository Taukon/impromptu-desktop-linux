### Install

```bash
(sudo apt install -y nodejs npm)
sudo apt install -y build-essential python3 python3-pip
sudo apt install -y libx11-dev libxtst-dev libjpeg-dev xvfb
(sudo apt install -y xterm firefox)
```
---

### Start

```bash
impromptu-desktop$ cd linux
impromptu-desktop/linux$ npm install
impromptu-desktop/linux$ npm run prod
impromptu-desktop/linux$ npm run start
```
